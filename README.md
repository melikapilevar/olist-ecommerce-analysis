# Olist E-Commerce Analysis

## Overview
This project includes exploratory data analysis (EDA), visualizations, and machine learning models to predict delivery times and analyze sales patterns of the Brazilian E-Commerce dataset by Olist.

## Dataset
The dataset used is the public Brazilian E-Commerce dataset by Olist from Kaggle:
https://www.kaggle.com/olistbr/brazilian-ecommerce

## Features
- Customer data
- Orders and order items
- Delivery times and shipping info

## Analysis
- Data cleaning and preprocessing
- Visualizations: Line charts, Heatmaps, Tree Maps, Histograms, Boxplots
- Machine Learning: Linear Regression, Decision Tree, Random Forest, Gradient Boosting
